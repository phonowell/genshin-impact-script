# 按键

所有按键采用游戏默认键位，但需要将`切换走/跑`绑定至`右Ctrl`。

| 按键                       | 功能                   |
| -------------------------- | ---------------------- |
| 长按`左键/侧键`            | 启用自定义动作         |
| `右键`                     | 退出当前菜单           |
| `Alt + F4`                 | 退出                   |
| `Ctrl + F5`                | 重新载入               |
| `F11`                      | 进入/退出自动钓鱼模式  |
| `F12`                      | 扫描小队               |
| `~`                        | 切换角色并使用元素战技 |
| `1~5`                      | 切换角色               |
| `Alt + 1~5`                | 切换角色并使用元素爆发 |
| `Alt + W`                  | 进入/退出自动前进模式  |
| `W / S`                    | 退出自动前进模式       |
| 长按`F`                    | 快速拾取物品/跳过对话  |
| `Alt + F`                  | 进入/退出自动拾取模式  |
| `Alt + Space`              | 启用/停用改良跳跃      |
| `Left/Right/Up/Down`       | 移动摄像机             |
| `Win + Left/Right/Up/Down` | 移动窗口               |
| `Ctrl + Numpad Dot`        | 开始/结束录像          |
| `Ctrl + Numpad 0~9`        | 播放录像               |
