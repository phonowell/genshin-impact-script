# GIS

中文 | [English](./doc/en/readme.md)

原神脚本，包含自动钓鱼、自动拾取、战斗辅助等多项功能。

基于`coffee-ahk`开发，体积小占用低。通过图像识别实现功能，不涉及任何内存读写或网
络封包修改。

**不包含任何作弊功能**。
